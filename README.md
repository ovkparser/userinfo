# OVK UserInfo

Веб-приложение (или сайт) для просмотра информации о пользователях OpenVK с удобным интерфейсом. Парсит информацию о пользователе OpenVK, но пока что только из основной инстанции. В будущем планируется поддержка разных инстанций.

Если у вас есть инстанция OpenVK, и вы хотите добавить её в парсер, свяжитесь с [@evilnyx](https://t.me/evilnyx) в Telegram. Как раз для этого пишется функционал.

## История появления проекта

Сидел значит в чате [Сейберпедии](https://saberpedia.no/) в [Telegram](https://t.me/sbrpdk). И один человек захотел сделать плеер музыки для OpenVK. И я такой: дай попробую поиграться с API, может помогу. И по приколу, случайно, был создан парсер публичной информации пользователя в OpenVK.

## Возможности

- Просмотр основной информации профиля
- Отображение статистики (друзья, подписчики, записи и т.д.)
- Просмотр контактов и социальных сетей
- Отображение личной информации
- Просмотр последних записей со стены
- Поддержка репостов и вложений
- Динамический favicon, отображающий состояние загрузки

## Технологии

- HTML5
- CSS3 (с поддержкой CSS Variables)
- JavaScript (ES6+)
- OpenVK API

## Установка и запуск

Если хотите запустить эту штуку локально, вот инструкция.

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ovkparser/userinfo.git
cd userinfo
```

2. Откройте `index.html` в браузере

Если же хотите открыть без скачивания чего-либо:

[Открыть сайт](https://ovkparser.github.io/userinfo/)

## Использование

1. Введите ссылку на профиль OpenVK в формате:
   - `https://ovk.to/id123`
   - `https://ovk.to/username`
   
2. Нажмите кнопку "Получить данные"
3. Дождитесь загрузки информации
